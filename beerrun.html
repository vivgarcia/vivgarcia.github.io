Overview

Beer Run was created using HTML, Bootstrap, CSS/SASS, JavaScript, jQuery/AJAX, Open Brewery API and Yelp API.

This project was a collaborative effort with a front-end and back-end team.

My role was the project manager and on the front-end team designing and coding the UI and ensuring proper communication with the back-end team.

How To Use:
  • Enter your desired city and state, type of brewery, tags or name of brewery then select 'Search'
  • Approximately 20 breweries will populate and three of their Yelp reviews
  • If there is no breweries available, a pop up will notify you to try again.
